## ğŸ”§ Troubleshooting

### Microphone Not Working
1. Check browser permissions (allow mic access)
2. Use Chrome or Edge (best support)
3. Ensure no other apps using mic
4. Try refreshing the page

### AI Not Responding
1. Check internet connection
2. Verify API key is configured
3. Look for error toasts
4. Try ending and restarting interview

### Voice Too Fast/Slow
1. Adjust **Speech Rate** slider
2. 1.0x = normal speed
3. 0.5x = half speed (slower)
4. 2.0x = double speed (faster)

### Poor Recognition Quality
1. Speak closer to mic
2. Reduce background noise
3. Speak more clearly
4. Check confidence score
5. If <60%, try again
